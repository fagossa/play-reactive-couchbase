Quick Overview

This project shows how to use the dependency injection mechanism in Play2.5 to communicate with a Couchbase Db.

Running

  • > sbt compile
  • > sbt test
  • > sbt run

Once the application has been compiled and the server started, your application can be accessed at: http://localhost:9000.
